ğŸ‘¤ Player Search

Search for a player by name to view their kill logs and statistics.

How to Search

  1. Make sure the "Player" tab is selected at the top
  2. Enter a player name in the search box (suggestions appear after 2 characters)
  3. Select from suggestions or press Enter to search
  4. To change servers, use the dropdown to the right of the search box
ğŸ’¡ Tip

Search history is saved automatically. Click the search box to see previously searched players.

ğŸ° Guild Search

Search for a guild to view guild information and member list.

How to Search

  1. Click the "Guild" tab at the top
  2. Enter a guild name in the search box
  3. Select from suggestions or press Enter to search

Guild Page Information

  • Basic Info - Founder, founding date, alliance
  • Statistics - Member count, Kill Fame, Death Fame, Fame Ratio
  • Member List - All members with fame data (sortable & filterable)
ğŸ’¡ Tip

Click a member's name to go to their kill log page.

View Modes

There are two view modes: "Individual Logs" and "Battles".

  • Individual Logs - Shows each combat event chronologically
  • Battles - Groups multiple events into battle summaries

Reading Log Cards

  • Left side - Your info (equipment, IP)
  • Center - Result icon (âš”ï¸/ğŸ’€), fame, time, location
  • Right side - Enemy info (equipment, IP)

  • Guild Statistics - Kill/death counts by guild
